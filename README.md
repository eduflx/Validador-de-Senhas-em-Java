# Validador-de-Senhas-em-Java
Programa em Java que reliza validações em uma senha digitada pelo usuário.
O Programa é baseado em 2 funções dentro da classe principal: 

*Validar tamanho: Verifica a quantidade de caracteres da senha. 
*Validar caracteres especiais: Verifica se a senha possuí ou não os caracteres especiais especificados.

O programa retorna os resultados dentro da classe principal, que pede a entrada da senha para o usuário e executa as funções.

Exercicio 3 da Lista de exercícios nº3 de Extrutura de Dados 1 do curso de ADS do IFTM Campus Patrocínio.
